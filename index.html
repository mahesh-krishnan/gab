What is Global Azure Bootcamp?

On, Saturday, April 16, 2016 we are out to set some records again!

In April of 2013 the first Global Windows Azure Bootcamp was held at more than 90 locations around the globe! In March 2015 it was topped with 182 locations! This year we are again doing a one day deep dive sessions to help thousands of people get up to speed on developing Cloud Computing Applications for Azure.

In addition to this great learning opportunity you have the opportunity to participate in the Global Lab - a horizontally scalable calculation challenge. This year's global charity lab challenge is still being worked out and details will be shared when available.

Download the Windows Phone Application - Global Azure BootCamp

Melbourne has hosted Azure bootcamp since 2013, and we've had sessions on Azure Websites, Mobile Services, Cloud Services and Azure IaaS. 

This year we continue with two tracks, an IT Pro Track, and a Developer Track you can take your pick between Developer topics and IT Pro focused topics all around Azure.
